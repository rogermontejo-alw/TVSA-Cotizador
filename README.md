============================================================ SISTEMA DE COTIZACIONES TELEVISA MID (v1.0) ================================================================Diseñado y desarrollado por Roger Montejo.Plataforma de gestión de tarifas de pauta utilizando React y Google Sheets como backend.[ I ] ARQUITECTURA Y CARACTERÍSTICAS FRONTEND: React 19 (Vite) + Tailwind CSS (Estilo Manrope/Poppins).BACKEND DE DATOS: Google Sheets (CSV Export) para Lectura.BACKEND DE ESCRITURA: Google Apps Script (Servicio Web API) para registrar Clientes y Condiciones.ALCANCE: Permite cotizaciones Multi-Plaza y gestión de paquetes VIX.[ II ] REQUISITOS DEL ENTORNO Necesitas Node.js (18+), npm/Yarn, y acceso de edición a Google Sheets.[ III ] CONFIGURACIÓN DE APPS SCRIPT (ESCRITURA)Obtener Script: Copia el contenido del archivo Code.gs en un nuevo script de tu Google Sheets (Extensiones > Apps Script).Despliegue: Despliega el script como "Aplicación web" con Acceso configurado a Cualquiera.URL API: Guarda la URL de despliegue que termina en /exec.Actualizar Frontend:Abra el archivo App.jsx y reemplace la URL de la constante:const SHEET_WRITER_API = 'TU_NUEVA_URL_DE_APPS_SCRIPT/exec';
